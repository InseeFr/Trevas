Introduction

Introduction

Trevas est un outil permettant de transformer et de valider des données statistiques grâce au langage VTL 2.0.

Description technique

Trevas fournit des moteurs VTL pour différents environnemens d'exécution, notamment un moteur Java 11 et un moteur Apache Spark.

Les moteurs Trevas se basent sur la spécification JSR 223 qui décrit l'utilisation de langages de scripts en Java.
