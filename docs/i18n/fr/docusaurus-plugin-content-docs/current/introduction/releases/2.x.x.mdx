## Version 2.3.0 - 04/05/26

(Voir la note technique sur [Github](https://github.com/InseeFr/Trevas/releases/tag/v2.3.0))

- Correction de l'implémentation Spark pour les identifiants contenant des points
- Correction de la grammaire pour autoriser les commentaires en fin de scripts

## Version 2.2.0 - 08/02/26

(Voir la note technique sur [Github](https://github.com/InseeFr/Trevas/releases/tag/v2.2.0))

- Améliorer la prise en charge des fonctions `keep`, `drop` et `rename` pour une meilleure conformité avec la spécification VTL.
- Améliorer l'implémentation du DAG en ajoutant la notion de portée.

## Version 2.1.0 - 27/12/25

(Voir la note technique sur [Github](https://github.com/InseeFr/Trevas/releases/tag/v2.1.0))

- Amélioration des exceptions Trevas pour encapsuler les exceptions Spark
- Correction de la possibilité d'insérer un point dans les noms de variables

## Version 2.0.0 - 28/09/25

(Voir la note technique sur [Github](https://github.com/InseeFr/Trevas/releases/tag/v2.0.0))

- Implémentation d'un DAG et réorganisation des instructions VTL avant exécution
- Correction des problèmes liés à `vtl-prov` et `vtl-sdmx`
- Correction d'un problème de comparaison
- Correction de l'initialisation des liaisons pour éviter la réutilisation des variables
