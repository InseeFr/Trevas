[Trevas](https://github.com/InseeFr/Trevas)是一个 Java 库，用于使用[VTL 2.0](https://sdmx.org/?page_id=5096)语言来转换和验证统计数据 .

## 技术说明
Trevas项目在各种运行环境下提供VTL引擎，包括 Java 11 引擎和 [Apache Spark](https://spark.apache.org/)引擎.

Trevas的引擎是基于[JSR 223](https://jcp.org/en/jsr/detail?id=223)规范，该规范描述了 Java 中脚本语言的使用
