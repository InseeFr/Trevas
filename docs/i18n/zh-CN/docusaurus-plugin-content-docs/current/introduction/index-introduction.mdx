[Trevas](https://github.com/InseeFr/Trevas)是一个 Java 库，用于使用[VTL 2.0](https://sdmx.org/?page_id=5096)语言来转换和验证统计数据 .

## 什么是VTL？

- 面向统计学家的商业、逻辑级和用户友好的语言
- 依赖于标准结构元数据
- 可互操作，与平台无关
- 函数式语言，支持数据沿袭以获得更多洞察力和可重复性

## 技术说明

Trevas 项目在各种运行环境下提供 VTL 引擎，包括 Java 11 引擎和 [Apache Spark](https://spark.apache.org/)引擎.

Trevas 的引擎是基于[JSR 223](https://jcp.org/en/jsr/detail?id=223)规范，该规范描述了 Java 中脚本语言的使用
