[Trevas](https://github.com/InseeFr/Trevas) is a Java library for transforming and validating statistical data using the [VTL 2.0](https://sdmx.org/?page_id=5096) language.

## What is VTL?

- Business, logical and user-friendly language for statisticians
- Relies on SDMX metadata
- Interoperable, execution platform agnostic
- Functional language, enabling data lineage for more understanding

## Technical description

Trevas provides VTL engines for different execution contexts, in particular a Java 11 engine and an [Apache Spark](https://spark.apache.org/) engine.

Trevas engines are based on the [JSR 223](https://jcp.org/en/jsr/detail?id=223) specification (Scripting for the Java Platform), which describes the use of scripting languages in Java.
