### Read

```java
Dataset<Row> sparkDataset = spark.read()
                                .option("delimiter", ";")
                                .option("header", "true")
                                .csv("folder_path");
SparkDataset dataset = new SparkDataset(sparkDataset);
```

See all supported options in the [official documentation](https://spark.apache.org/docs/latest/sql-data-sources-csv.html#data-source-option).

### Write

The CSV format is not recommended for writing Trevas `Dataset`s (see <a href={useBaseUrl('/developer-guide/spark-mode/data-sources#bonnes-pratiques-trevas')}>here</a>)
