### Trevas good practices

The <a href={useBaseUrl('/developer-guide/spark-mode/data-sources/parquet')}>Apache Parquet</a> format is the only one allowing to store and manage VTL metadata when the Trevas engine is instantiated in Spark mode.

It is thus strongly recommended to use this format.
